Projects


Simpletheme.myshopify.com Partial Clone Project

(DevMountain - July 2018)

For my personal project, I created a partial clone of the e-commerce website Simpletheme.myshopify.com. On my website, a user can login, navigate between different categories of clothing, search for products by their description, view individual product details, add items to their cart, adjust the quantity of each item they are purchasing, and check out.

Technologies used: HTML5, CSS flexbox, JavaScript (ES6), React, Redux (for state management) Axios (for http requests), Auth0 (for login authentication), NodeJS, Express, Massive, and PostgresQL.

(Website | Github Repo)

'Rumblt' - Tumblr.com Inspired Group Project

(DevMountain - July 2018)

We created the full-stack 'Rumblt' website to loosely mimic the website Tumblr.com. With 'Rumblt', users can log in, create blog posts, follow other bloggers on the site, and like other blogger's posts. For this project, I was primarily responsible for working as one of the two developers in charge of the front-end (including the wireframing of the website itself).

Technologies used: HTML5, CSS flexbox, JavaScript (ES6), React, Redux (for state management) Axios (for http requests), NodeJS, Express, Massive, PostgresQL, Jest (for endpoint tests), and Cypress (for end to end tests).

(Website | Github Repo)
